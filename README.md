# Kontena Cloud Plugin

This plugin adds [Kontena Cloud](https://kontena.io/cloud) management commands to Kontena CLI.

## System Requirements

- [Kontena CLI](https://kontena.io/)

## Installation

```
$ kontena plugin install cloud
```
